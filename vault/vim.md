## Shorcuts

| Shortcut  | Description                         |
| --------- | ----------------------------------- |
| `>>`      | Indent line by shiftwidth spaces    |
| `<<`      | De-indent line by shiftwidth spaces |
| `5>>`     | Indent 5 lines                      |
| `5==`     | Re-indent 5 lines                   |
| `<C-w-w>` | Cycle through windows               |
| `<C-/>`   | Comment selection/line              |
| `<C-Q>`   | Save all and quit                   |
| `<C-s>`   | Save                                |
| `<C-n>`   | Open Vim-tree                       |
